HTML

HTML (HyperText Markup Language) is the foundation of web pages. It uses tags to structure content like text, images, and links, telling browsers how to display them.

CSS

CSS (Cascading Style Sheets) is like the makeup and clothing for your website.It's what makes it look good! While HTML provides the structure, CSS lets you control the style .It's essential for making websites visually appealing and user-friendly.  

Javascript

JavaScript is the magic ingredient that makes websites interactive. it's a programming language that adds dynamic behavior, like responding to clicks, validating forms, and updating content without reloading the page. 1 Think of it as the brains behind the website's actions, making it more engaging and user-friendly.  

React.js

Bootstrap
